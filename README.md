# Aka

AKA is an alias manager. keep tabs on all of your alias'ed commands, shortcuts, and more without hand-editing dot files.

## Installation

Add this line to your application's Gemfile:

    gem 'aka'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install aka
